# Inlover-html-javascript-php-web
เป็นโปรเจ็คเก่าตอนเริ่มเขียนโค้ดโดย ไม่ใช้เฟรมเวิร์คอะไรเลยทั้งสิ้น ตอนนั้นพึ่งเริ่ม ใช้แบบเพียวๆ เลยครับ ตอนปี 2566
ตอนนั้นร่วมทำกับเพื่อนโดยทำส่วน ของ backend และ api ส่วนของ frontend ทำนิดหน่อย

โดยเว็บจะเป็นเว็นแอพพลิเคชัน จับคู่ โดยมีระบบปัดการ์ด ระบบแชท ระบบจัดการโปรไฟล์ และอื่นๆ เป้นหลัก
